### Back-End of a "full-featured" collaborative text editor

> Multi role authentication : Owner, Editor, Viewer

> Password Reset flow

> Web sockets for collaboration (socket.io)

> in-memory data caching (Redis)  

##### Tech stack:

> Nodejs/Expressjs

> PostgreSQL

> GraphQL - apollo-server-express
